使用RedisTemplate操作Redis

使用RedisTemplate操作Redis

  • 补充：List,Set,Map 区别
    • List,Set都继承自Collection接口，Map不是
    • List特点：元素有放入顺序，元素可以重复
    • Set特点：元素无放入顺序，元素不可重复，重复元素会被覆盖。
      • (元素虽然放入无顺序，但是元素在set中的位置是由该元素的HashCode决定的，其位置其实是固定的)
    • Map特点：适合存储键值对的数据
  • Redis配置：spring-redis.xml
