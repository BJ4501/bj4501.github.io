UDP与JavaUDP操作

UDP是什么

  • 全称 User Datagram Protocol
  • 一种用户数据报协议，又称用户数据报文协议
  • 是一个简单的面向数据报的传输层协议，正式规范为RFC 768
  • 用户数据协议、非连接协议

UDP为什么不可靠

  • 它一旦把应用程序发给网络层的数据发送出去，就不保留数据备份
  • UDP在IP数据报的头部，仅仅加入了复用和数据校验（字段）
  • 发送端生产数据，接收端从网络中抓取数据
  • 结构简单、无校验、速度快、容易丢包、可广播
